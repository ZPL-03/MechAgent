# 结构静力验证知识

MechAgent 当前验证闭环覆盖线弹性结构静力问题，包括悬臂梁端点集中力、悬臂梁均布线载荷、四边简支矩形薄板均布压力、固支长方体端面压力和固支长方体端面合力。

悬臂梁端点集中力采用 Euler-Bernoulli 梁理论作为参考。端点挠度为 `P L^3 / (3 E I)`，其中 `P` 为端点力，`L` 为梁长，`E` 为弹性模量，`I` 为截面惯性矩。

悬臂梁全跨均布线载荷采用 Euler-Bernoulli 梁理论作为参考。端点挠度为 `q L^4 / (8 E I)`，其中 `q` 为单位长度载荷。

固支长方体轴向拉伸采用一维线弹性轴向位移作为参考。端面位移为 `F L / (E A)`，其中 `F` 为轴向合力，`A` 为横截面积。

矩形薄板均布载荷弯曲采用薄板小挠度理论参考，中心挠度由材料弹性模量、泊松比、板厚、边长和均布压力共同决定。验证时使用明确的参考值、相对误差和容许误差判断结果可信度。

求解结果报告应区分三类状态：求解器是否运行成功、存在参考值时工程校核是否通过、缺少参考值时是否属于未验证工况。
