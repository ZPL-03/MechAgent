# CalculiX 结构静力工作流

CalculiX 在 MechAgent 中作为开源有限元求解后端使用。框架生成结构静力输入文件，调用配置中的 `ccx` 可执行文件，读取求解输出，再将位移标量和验证指标传回编排层。

当前内置网格器生成可直接用于求解的结构网格输入。梁和长方体使用六面体单元，矩形薄板使用壳单元。网格器记录单元数、节点数、最小质量指标和网格尺寸，编排层根据这些结构化字段执行后续求解。

求解适配器只接受当前模型契约支持的线弹性静力任务。边界条件、载荷方向、材料参数、几何尺寸和分析类型在写入求解文件前完成校验。无法映射到当前契约的任务应返回明确错误，而不是生成不可靠输入文件。

后处理读取求解结果中的位移量，并按任务类型提取端点挠度、中心挠度或轴向位移。评价器根据任务绑定的参考公式和容许误差输出 `passed` 与 `verification_status`。

Agent 间通信通过 Pydantic v2 模型传递。Planner 输出任务意图，Designer 输出模型参数，MeshAgent 输出网格结果，SolverAgent 输出求解摘要，PostProcAgent 输出后处理摘要，AnalystAgent 输出工程解释，ReporterAgent 汇总 Markdown 报告。
